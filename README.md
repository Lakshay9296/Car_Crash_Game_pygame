# Car Crash Game

Welcome to the **Car Crash Game**! This is a Python-based game created using Pygame where players navigate a car through a challenging course while avoiding obstacles and crashes.

## Features

- **Dynamic Gameplay**: Control a car and navigate through various obstacles.
- **Collision Detection**: Realistic crash mechanics and obstacle interactions.
- **Score Tracking**: Keep track of your score based on performance and distance traveled.

## Technologies Used

- **Python**: Programming language used for development
- **Pygame**: Library for creating video games

## Gameplay

- Use arrow keys to navigate the car.
- Avoid obstacles and crashes to achieve a high score.
